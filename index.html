Research Experience

My research interests include leveraging interdisciplinary technologies to automate mundane tasks in the construction industry.

Completed Research

Virtual Reality Framework for Machine Learning Enhanced Carbon Footprint Reduction in Construction industry

While BIM does enable preconstruction evaluations, it cannot visualize embodied carbon data seamlessly. To bridge this gap, a virtual environment in Unity was built to combine the functionalities of Autodesk Revit with the ML based web application. Several supervised ML models, such as AdaBoost, Random Forest, and Linear Regressions, were trained on the sampled concrete mixture datasets. However, the XGBoost algorithm performed the best with more than 90% accuracy. Then, the outputs were exported as a web-based application. Finally, the virtual environment could be accessed using a provided link.

Supervisor(s): Dr. Md. Ikramul Hoque

A Systematic Review of Deep Learning-Based Drone Applications in the Construction Industry

The review followed PRISMA guidelines to select and analyze studies, looking for patterns and themes in the research. Significant improvements were found in the use of Convolutional Neural Networks (CNNs) and You Only Look Once (YOLO) algorithms for detecting objects, monitoring sites, and assessing structures. These technologies have proven to be highly accurate and efficient in different phases of construction.

Supervisor(s): Dr. Md. Habibur Rahman Sobuz

Projects

BIM Collaboration: Revit-Based Architecture and Plumbing

This project demonstrates a unified architectural and plumbing design developed in Revit for a commercial building. The 3D model leverages parametric components to ensure accurate hot/cold water supply, drainage, and fixture connections. Each deliverable highlights carefully coordinated routes, pipe sizes, and equipment placements. Legacy data was cross-referenced using AutoCAD, ensuring seamless project coordination.

Software used: Autodesk Revit, AutoCAD

HVAC System Design and BIM Coordination in Autodesk Revit

Showcasing an HVAC system modeled and coordinated in Autodesk Revit. The 3D design aligns ducting, equipment, and mechanical components while ensuring proper flow rates. Accurate construction drawings highlight dimensioned routes, flow capacities, and device placements. Interdisciplinary BIM coordination and parametric workflows guarantee system efficiency and reduced on-site clashes.

Software used: Autodesk Revit

Clash Detection and Coordination using Navisworks

This project demonstrates the use of Navisworks and Revit for clash detection and model coordination. By integrating architectural, structural, and MEP models, potential conflicts were identified early in the design phase, minimizing errors during construction. The 3D clash detection process ensured a streamlined workflow, improving project accuracy and reducing costly rework.

Software used: Autodesk Navisworks
